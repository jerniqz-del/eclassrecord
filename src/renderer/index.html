    Backup Directory Path
    None configured. Select a directory (like your OneDrive or Google Drive folder) to save duplicate database copies and daily rolling backups (up to 30 days) in both directories.

    Grading Preferences

    Auto-blur Grades
    Automatically blur grades and scores when loading grading sheets to hide them from spectators.
    Show Numerical Equivalents (DO 15 s. 2026 Annex C)
    Show numerical grade range equivalents (e.g. 90-100 for A) next to descriptive letters in tables, reports, and exports.
    Use Universal Trimester Layout
    ⚠ Caution: Enabling this will apply the Key Stage 2 Trimester calculation/layout (5 WW, 3 PT, 3 SA/TE columns) to all grade levels. It will reconstruct columns for existing class records, resetting mismatching scores.

    👋 Welcome to E-Class Record

    Your local, secure, and teacher-owned grading assistant. Here are the core features of the application:

    Roster Management
    Manage class workloads, add students, and import rosters directly from official DepEd SF1 spreadsheets.
    DepEd Grading Calculations
    Calculates terms using standard DO 15 s.2026 transition, zero-based, or Key Stage 2 Trimester models.
    Quick Grade Entry
    Sequential grade entry wizard. Type a score and press Enter to save and advance automatically.
    Backups & Export
    Download secure offline backups of your database, print class records directly, or export premium PDF reports.
